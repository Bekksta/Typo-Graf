# Typo Graf

[English](README.md) · **Русский**

Плагин для Figma, который приводит текст в макетах к типографическим нормам — расставляет неразрывные пробелы, кавычки, тире, степени, единицы, чинит апострофы и ёфикацию. Семь языков, защита URL и email, сохранение стилей, оффлайн.

## Почему он

- **Лёгкий.** Бандл < 1 МБ — вместе со встроенным словарём ёфикации на ~107 000 словоформ. Никаких UI-фреймворков, заставок и онбордингов.
- **Без задержек.** Открывается мгновенно: прогресс-бар и кнопка «Отменить» — больше ничего на экране.
- **Оффлайн.** Все правила, словари и логика — в бандле. `networkAccess: none` в манифесте, ничего не уходит наружу.
- **Сохраняет стили.** `bold`, `italic`, шрифты, размеры и цвета остаются на месте — правки идут через диапазоны, а не перезаписью текста узла.
- **Не трогает URL и email.** Они защищаются равной длины маской на время прохода правил.
- **Один прогон — одно undo.** Все изменения откатываются обычным `Ctrl/⌘+Z`.
- **Тык — и работает.** Без настроек, без диалогов, без выбора режима.

## Как пользоваться

1. **Plugins → Typo Graf** в меню Figma. Если что-то выделено — плагин обработает текстовые узлы внутри выделения (включая вложенные). Если ничего не выделено — пройдётся по всем текстовым узлам текущей страницы.
2. Дождитесь окончания (или нажмите «Отменить»).
3. По завершении плагин показывает сводку: сколько изменений, на скольких узлах, по каким языкам.

Лимит за один запуск — 2000 узлов. Узлы текстом длиннее 5000 символов пропускаются и попадают в итоговую сводку.

## Языки

Определяются автоматически на уровне каждого текстового слоя — по скрипту и характерным диакритикам:

| Язык | Маркеры |
|------|---------|
| `ru` | кириллица (фолбэк) |
| `uk` | `і ї є ґ` |
| `en` | латиница (фолбэк) |
| `fr` | `ç é â æ œ ÿ` |
| `de` | `ä ö ü ß` |
| `es` | `ñ á í ó ú ¿ ¡` |
| `bcs` | `č ć š đ ž` (`bs`/`hr`/`sr` латиница) |

Сербская кириллица обрабатывается как русский.

## Правила

### Общие (все языки)

- `...` → `…` (U+2026), лишние пробелы вокруг чистятся.
- `10-12` → `10–12` (en dash для числовых диапазонов).
- Двойные пробелы → одинарный.
- `12 кг`, `20 %`, `300 ₽` → NBSP между числом и единицей/валютой/процентом.
- `12''` → `12″`, `12'` → `12′` (праймы).
- `45 deg` → `45°`.

### Математика

Работает на инлайн-выражениях, не лезет в LaTeX/MathML-блоки.

- **Степени:** `x^2 → x²`, `y^10 → y¹⁰`, `10^-3 → 10⁻³`, `e^(x) → eˣ`, `(a+b)^n → (a+b)ⁿ`.
- **Индексы:** `x_1 → x₁`, `log_10(x) → log₁₀ x`.
- **Умножение:** `a*b → a · b`, цепочки `a*b*c → a · b · c`. Жирный (`**...**`) не трогается.
- **Деление:** `a/b → a / b`, `1/2 → ½`, `(a+b)/(c+d)` без замены `/` на `÷`.
- **Сравнения:** `!= → ≠`, `<= → ≤`, `>= → ≥`, `= → =` (с пробелами), пробелы вокруг `≈ ≃`.
- **Знаки:** `+- → ±`, `-+ → ∓`.
- **Стрелки:** `-> → →`, `<- → ←`, `--> → ⟶`, `=> → ⇒`, `<=> → ⇔`.
- **Константы и функции:** `pi → π`, `sqrt(x) → √x`, `inf → ∞`, `\alpha → α` (и остальные греческие), `sin(x) → sin x`, `log10(x) → log₁₀ x`, `lim(x→0) → limₓ→₀`, `vec(a) → a⃗`.

### Русский (`ru`)

- NBSP после коротких предлогов и союзов: `в дом`, `на улице`, `и т. п.`
- NBSP перед частицами `бы`, `ли`, `же`.
- Инициалы: `А. С. Пушкин` — NBSP между инициалами и фамилией.
- Аббревиатуры с точкой: `г. Москва`, `ул. Ленина`, `№ 8`, `§ 104`, `1981 г.` — NBSP после.
- Составные сокращения: `и т. д.`, `т. е.`, `до н. э.` — нормализованы вместе с NBSP.
- Дефисные сокращения без точки: `г-н`, `г-жа`, `д-р`, `р-н` — неразрывный дефис (U+2011) внутри и NBSP после.
- Кавычки-ёлочки `«…»`. Пунктуация прижимается к закрывающей.
- Тире: двойной дефис `--` и одиночный `-` между не-цифровыми токенами → `—`, схема `слово<NBSP>—<space>слово`.
- **Ёфикация на ~107 000 словоформ** из словаря [eyo-kernel](https://github.com/e2yo/eyo-kernel). Заменяются только «безопасные» формы — те, где нет омонима без `ё`. То есть `ребенок → ребёнок`, `учет → учёт`, `шел → шёл`, но `все` остаётся как есть (омоним множ. числа от «весь»). Случайных правок не делаем.

### Английский (`en`)

- Smart quotes: `"..."` → `“…”`, вложенные `'...'` → `‘…’`.
- Апострофы в словах и на конце: `don't` → `don’t`, `lovers'` → `lovers’`.
- Праймы: `12''` → `12″`, `12'` → `12′`.
- `--` → `—` без навязывания пробелов вокруг (по исходному стилю).
- Диапазоны: `10-12 km` → `10–12 km`.
- Валюты: `$1234.5` → `$1,234.5` (US-формат). Порядок `$ 300` / `300 $` сохраняется, NBSP добавляется.
- NBSP после служебных слов (`a`, `an`, `the`, `and`, `but`, `or`, `to`, `of`, `in`, `on`, `at`, `by`, `as`, `I`).
- NBSP перед единицами: `15 km`, `20 %`.

### Французский (`fr`)

- Гийеметы `« … »` с узкими NBSP (U+202F) внутри.
- Узкий NBSP перед `; : ? ! »`.
- Число + единица/валюта/процент → узкий NBSP.

### Украинский (`uk`)

- Кавычки `«…»`.
- NBSP после коротких предлогов: `в`, `у`, `з`, `із`, `й`, `та`, `а`, `і`.
- Узкий NBSP после `№`, `§`, `стор.`, `рис.`, `м.`.
- Число + единица/валюта/процент → узкий NBSP.

### Немецкий (`de`)

- Кавычки `„…“`. Если в тексте уже стоит `»…«` — не трогает.
- NBSP перед единицами и валютой.
- Десятичная запятая не трогается.

### Испанский (`es`)

- Кавычки `“…”` по умолчанию. Если уже стоит `«…»` — не трогает.
- NBSP перед единицами и валютой.
- `¿` и `¡` остаются как есть, искусственных пробелов вокруг `:;!?` не добавляется.

### BCS (`bs`/`hr`/`sr` латиница)

- Кавычки нормализуются по встреченному варианту (`«…»` или `„…“`). По умолчанию — `„…“`.
- NBSP перед единицами и валютой.

## Что плагин НЕ делает

- **Не трогает URL и email.** Маскируются равной длины placeholder'ами, после прохода восстанавливаются обратно.
- **Не ломает переносы строк.** `\n` сохраняется, склейка через перенос не делается.
- **Не лезет в LaTeX/MathML-блоки.** Только короткие inline-операнды.
- **Не редактирует символьное выделение внутри узла.** Только текстовые узлы целиком.
- **Не ёфицирует слова с омонимами.** Чтобы не превратить «все» в «всё» там, где имеется в виду «весь».
- **Не подключает кастомные шрифты.** Узлы с шрифтом, недоступным в Figma, пропускаются и попадают в итоговый счётчик.

## Интерфейс

Окно 320×160 px:

- Прогресс-бар.
- Текущая операция (`Сканирование…`, `Обработка…`, `Отмена…`).
- Счётчик обработанных узлов: `X / Y`.
- Кнопка «Отменить» — останавливает после текущего узла; всё, что уже применилось, остаётся (откатывается общим `Ctrl/⌘+Z`).

Финальная сводка локализована под язык Figma:

```
Готово. Языки: русский, английский. Изменений: 142. Затронуто узлов: 38.
Ctrl/⌘+Z — отменить
```

## Локализация

UI плагина и финальный notify локализованы на семи языках, по `navigator.language` Figma:

| Локаль | Соответствие `navigator.language` |
|--------|-----------------------------------|
| Русский | `ru-*` |
| English | `en-*` и любая нераспознанная локаль (фолбэк) |
| Français | `fr-*` |
| Українська | `uk-*` |
| Deutsch | `de-*` |
| Español | `es-*` |
| BCS (Bosnian / Croatian / Serbian latin) | `hr-*`, `sr-*`, `bs-*` |

## Установка (для разработки)

```bash
npm install
npm run build       # одноразовая сборка
npm run dev         # сборка в watch-режиме
```

Затем в Figma desktop → **Plugins → Development → Import plugin from manifest** → выбрать `manifest.json` в корне репозитория.

## Тесты

```bash
npm test            # один прогон
npm run test:watch  # watch-режим
npm run showcase    # прогон по корпусу с before/after-выводом
```

После `npm test` подробная диагностика лежит в `test-results/report.json`: по каждому правилу — входной текст, ожидаемый/полученный результат и точные позиции расхождений с код-пойнтами Unicode. `npm run showcase` печатает наглядные before → after по реалистичному корпусу с подсветкой невидимых символов.

## Ограничения

- Селекция работает на уровне узлов целиком, а не отдельных символов внутри узла.
- Лимит — 2000 текстовых узлов за один запуск; узлы длиннее 5000 символов пропускаются.
- Узлы с недоступными в Figma шрифтами пропускаются.
- Откат — нативным `Ctrl/⌘+Z` (общая undo-группа на весь прогон).
